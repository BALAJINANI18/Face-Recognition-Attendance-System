# 🚀 Face Recognition Attendance System

This project is a **Face Recognition Based Attendance System** built using Python and DBMS concepts. It automatically detects faces and marks attendance in real-time.

---

## 📌 Features

- 📸 Face detection and recognition
- 🧠 Real-time attendance marking
- 🗄️ Database integration using SQLite
- ❌ Prevents duplicate attendance using constraints
- 🎯 Simple and efficient system

---

## 🛠️ Tech Stack

- Python
- OpenCV
- face_recognition
- SQLite (DBMS)
